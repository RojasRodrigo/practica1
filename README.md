# Compilación y ejecución

## Compilación
Ejecuta el siguiente comando en la carpeta del proyecto para compilar la aplicación:

```bash
dotnet build
```
## Ejecución
### en Windows
```bash
.\bin\Debug\net6.0\MathOperationsApp.exe

```
### en Linux o MacOS
```bash
./bin/Debug/net6.0/MathOperationsApp

```
